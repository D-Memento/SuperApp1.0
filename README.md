 SuperApp — файловый менеджер для Linux

SuperApp — графический файловый менеджер на Java с быстрым доступом к документам, корзине, системной папке и подключаемым устройствам.

 Что умеет программа

- Древовидная навигация по разделам:
  - Мои документы
  - Корзина
  - System
  - Съёмные устройства
- Карточный просмотр содержимого папок в правой панели.
- Поиск файлов и папок по имени.
- Контекстные действия:
  - создать файл/папку,
  - открыть,
  - копировать/вставить,
  - переименовать,
  - удалить в корзину,
  - восстановить из корзины,
  - очистить корзину,
  - посмотреть свойства.
- Drag & Drop для работы с файлами.
- Горячие клавиши (в разделе «Справка»):
  - `Ctrl + N` — создать файл,
  - `Ctrl + D` — удалить файл,
  - `Ctrl + R` — восстановить,
  - `Ctrl + T` — очистить корзину,
  - `F1` — информация о программе.
- Встроенный терминал и всплывающее окно с системной информацией.
- Пункт «Устройства» для ручного обновления списка подключённых носителей.


 Быстрый запуск на Linux

 1) Перейти в каталог проекта

```bash
cd /путь/к/SuperApp
```

 2) Запустить программу
```bash
./run-superapp.sh
```

Скрипт выполняет прямой запуск приложения без дополнительных проверок:

```bash
java -jar SuperApp.jar
```

 Запуск без скрипта (альтернатива)

```bash
cd /путь/к/SuperApp
java -jar SuperApp.jar
```
---

 Где хранятся рабочие папки

При старте рядом с приложением используются/создаются каталоги:

- `Мои документы`
- `Корзина`
- `System`

Это удобно для локальной работы и тестирования, так как не затрагивает системные каталоги пользователя.

---

 Полезные замечания

- Для некоторых системных утилит из меню (например, `gnome-system-monitor`) нужна соответствующая среда рабочего стола и установленные пакеты.
- Если утилита недоступна, программа покажет сообщение об ошибке — это нормально.
